About Barricade-Angular

Barricade-Angular provides frontend access to the Barricade Web API.

As its name implies, Barricade-Angular requires Barricade and Angular.

Installation

In addition to downloading the source code on github, you can also install Barricade-Angular using Bower:

bower install barricade-angular
